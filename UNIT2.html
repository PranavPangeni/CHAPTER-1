1. Working with a Function

a. Define Function

A function in C is a block of code that performs a specific task, which is grouped together under a name and can be invoked from other parts of the program. It improves modularity, reusability, and maintainability of code.

b. Syntax of Functions

return_type function_name(parameter_list) {
    // Function body
    return value;
}

c. Types of Functions

  • Library Functions: Pre-defined in C libraries, e.g., printf(), scanf().
  • User-defined Functions: Created by the programmer to perform custom tasks.

d. Components of Function

  • Function Prototype: Declares the functionâ€™s signature before usage.
  • Function Call: Invocation of the function within main() or another function.
  • Function Definition: Complete implementation of what the function does.
  • Return Type: Specifies the type of value returned, or void for none.

2. Categories of Function with Examples

  • With Return Type, No Arguments: 
    int giveNumber() {
    return 100;
}
  • With Return Type, With Arguments: 
    int sum(int a, int b) {
    return a + b;
}
  • No Return Type, No Arguments: 
    void greet() {
    printf("Hello\n");
}
  • No Return Type, With Arguments: 
    void showSquare(int x) {
    printf("Square is %d", x * x);
}

3. Storage Classes

  • Automatic (auto): Default for local variables; lifetime limited to block; not explicitly required.
  • External (extern): Declares global variables accessible across files.
  • Register: Stored in CPU registers for faster access; limited in size and number.
  • Static: Retains value between function calls; lifetime is the entire program.

4. Recursive Function with Syntax and Examples

A recursive function is a function that calls itself. It requires a base condition to terminate recursion and prevent infinite loops.

int factorial(int n) {
    if (n == 0) return 1;
    else return n * factorial(n - 1);
}
int fibonacci(int n) {
    if (n == 0) return 0;
    else if (n == 1) return 1;
    else return fibonacci(n - 1) + fibonacci(n - 2);
}

5. Passing Array to a Function

Arrays in C are passed to functions by reference. This means any modifications made inside the function reflect in the original array.

Example 1: Printing Elements

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
}

Example 2: Sum of Array Elements

int sumArray(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}
