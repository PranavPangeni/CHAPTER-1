Unit 4: C Programming

Functions

Introduction and Syntax

A function in C is a block of code designed to perform a specific task. It improves modularity and code reusability.

return_type function_name(parameter_list) { // function body }

Purpose and Advantages

  • Reusability of code
  • Better readability and maintainability
  • Helps in debugging and testing
  • Reduces code duplication

Components of a Function

  • Function Prototype
  • Function Definition
  • Function Call
  • Return Statement

Function Prototype

int add(int a, int b);

Function Definition

int add(int a, int b) { return a + b; }

Function Call

int result = add(5, 10);

Types of Functions

  • Library Functions (e.g., printf, scanf)
  • User-Defined Functions

Passing Arguments

Arguments can be passed in two ways:

  • Call By Value – Copies value into function.
  • Call By Reference – Passes address, changes affect original variable.

Variable and its Scope

  • Local – Declared inside function, accessible only within it.
  • Global – Declared outside all functions, accessible everywhere.

Storage Classes

  • Automatic – Default, created inside function.
  • External – Declared with extern, global visibility.
  • Static – Retains value between function calls.
  • Register – Stored in CPU registers for faster access.

Function with Array Example

void display(int arr[], int n) { for(int i=0; i<n; i++) printf("%d ", arr[i]); }

Recursive Function

A function that calls itself.

int factorial(int n) { if(n==0) return 1; else return n * factorial(n-1); }

Advantages

  • Simplifies complex problems
  • Supports modular programming

Disadvantages

  • Consumes more memory
  • Slower due to repeated calls

Structure and Union

Structure: Introduction and Syntax

struct Student { int id; char name[50]; };

Structure Size

Depends on members and memory alignment.

Accessing Members

struct Student s1; s1.id = 1; strcpy(s1.name, "Ram");

Nested Structure

struct Address { char city[20]; }; struct Student { int id; struct Address addr; };

Array of Structures

struct Student s[10];

Passing Structure to Function

void display(struct Student s) { printf("%d %s", s.id, s.name); }

Union: Introduction and Syntax

union Data { int i; float f; char str[20]; };

Comparison between Structure and Union

  • Structure: All members have separate memory.
  • Union: All members share same memory.

Pointers

Introduction and Syntax

int *p; int a = 5; p = &a;

Usage and Concept

Pointers store the address of variables, enabling direct memory access.

Pointer and Function

void update(int *x) { *x = *x + 1; }

Call by Reference

update(&a);

Pointer with Arrays

int arr[5] = {1,2,3,4,5}; int *p = arr;

Advantages

  • Efficient memory usage
  • Supports dynamic memory allocation

Disadvantages

  • Complex syntax
  • Risk of dangling pointers

File Handling

Concept and Need

Files store large data permanently, unlike variables.

File Handling Functions

  • fopen(), fclose() – Open/close file
  • getc(), putc() – Read/write single character
  • fprintf(), fscanf() – Formatted I/O
  • getw(), putw() – Read/write integers
  • fgets(), fputs() – String I/O
  • fread(), fwrite() – Block I/O
  • remove(), rename() – File operations

Random Access Functions

  • fseek() – Move file pointer
  • rewind() – Move to start
  • ftell() – Return current position

File Opening Modes

  • r, r+ – Read
  • w, w+ – Write
  • a, a+ – Append

Steps to Work with File

  1. Define File Pointer
  2. Open File in Mode
  3. Perform Read/Write/Append
  4. Close File

End of File (EOF)

Special condition that indicates no more data is available for reading.
